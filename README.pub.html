DOWNLOAD MetaPost examples

The document mpdemo.pdf contains over 600 MetaPost diagram examples.
These MetaPost examples demonstrate a wide range of MetaPost's capabilities.
Most of these illustrations are used in my differential geometry book.
These examples are not a tutorial, but you could use them as a MetaPost cookbook.
To create your own diagrams, find the diagram in the mpdemo.pdf file which is closest to what you want.
Then look at the corresponding MetaPost diagram source to see how it is done.
To compile the PDF file mpdemo.pdf on a unix system from source, type the following. 
    make -f makefile.pub
